# fteepee - simple FTP client
fteepee is a simple FTP client written in Java using JavaFX technology

### Run & development
To run this application JVM 11 & JavaFX are required.

- First, check JAVA_HOME set properly
- Then, run:
```shell
./gradlew run
```
